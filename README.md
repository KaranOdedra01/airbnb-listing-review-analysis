# Airbnb Listings & Reviews Analysis

## Overview

This project presents an interactive Tableau dashboard created using Airbnb listings and review datasets. The dashboard helps explore important insights related to pricing, availability, customer feedback, and listing performance through visual analytics.

## Objectives

* Analyze Airbnb listing trends
* Understand customer review patterns
* Compare pricing across different listings
* Identify highly rated properties
* Visualize important business insights using dashboards

## Tools & Technologies

* Tableau
* Data Visualization
* Data Analysis

## Dashboard Features

* Pricing analysis
* Review and rating insights
* Availability trends
* Listing comparison visuals
* Interactive filters and charts

## Project Files

* `AirBnB_Project.twbx` – Tableau packaged workbook containing dashboards, worksheets, and dataset connections.

## Key Insights

The dashboard highlights how pricing, reviews, and availability influence Airbnb listings. It also helps identify popular properties and customer behavior patterns using clear and interactive visualizations.

## How to Use

1. Download the `.twbx` file
2. Open it in Tableau Desktop or Tableau Public
3. Explore the dashboard using filters and charts

## Conclusion

This project demonstrates how data visualization can convert raw Airbnb data into meaningful insights that support better understanding and decision-making.
